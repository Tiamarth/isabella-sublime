## Install

Put the "Isabella Color Scheme" folder in your Packages folder.

**Sublime Text 3**
```shell
git clone https://github.com/Tiamarth/Isabella.git && cd Isabella/sublime
mv Isabella Color Scheme ~/.themes/.config/sublime-text-3/Packages
```
